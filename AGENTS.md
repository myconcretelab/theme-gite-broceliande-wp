# Instructions pour les agents

- Apres chaque changement livre, augmenter tres legerement la version du plugin afin de forcer le nettoyage du cache. Utiliser une incrementation minimale, par exemple `1.5` vers `1.5.1` ou `1.5.1` vers `1.5.2`, sans changer la numerotation plus que necessaire.
- Les donnees utilisees par les blocs ne viennent pas de ce theme/plugin directement. Elles sont fournies par le repo `contrats`, situe a cote de l'installation WordPress.
