== Description ==

Theme WordPress block/FSE pour les gites Broceliande.

Le theme est base sur Twenty Twenty-Five et contient des styles, templates et patterns adaptes au site.

== Installation ==

1. Placer le dossier dans `wp-content/themes/theme-gite-broceliande-wp`.
2. Activer le theme `Gites Broceliande` dans WordPress.

== Developpement ==

Installer les dependances :

`npm install`

Compiler le CSS minifie :

`npm run build`

Surveiller les changements CSS :

`npm run watch`
